# Discord Structure Template

- `#ops-command` — owner requests and approvals
- `#guest-inquiries` — intake and recommendations
- `#booking-desk` — booking decisions and updates
- `#guest-comms` — outbound/inbound communication summaries
- `#pricing-harness` — market pricing run outputs
- `#ops-log` — automation and health logs
- `#handoff` — staff handoff notes

This structure is portable to Slack/Telegram/other apps.
