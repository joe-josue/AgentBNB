# Runbook — Inquiry Recommendation Loop

1. Receive inquiry payload.
2. Validate required fields.
3. Check availability/calendar conflicts.
4. Review house rules and occupancy fit.
5. Send holding message to guest.
6. Submit recommendation: approve/reject/escalate with rationale.
7. Wait for owner action.
8. On decision, update booking and send appropriate guest message.
9. Record context in SoR + staff handoff if booked.
