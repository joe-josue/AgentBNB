# Media Checklist

Use this folder to track property photos, videos, and media references.

Do not commit private or high-resolution originals to a public repo unless you intend to publish them.

## Required Launch Assets

- hero image
- exterior/front image
- main amenity image
- sleeping area images
- kitchen image
- dining/common area image
- bathroom image
- location/access image, if relevant
- logo or mark, if available

## Recommended Naming

Use descriptive filenames:

```text
hero-pool-wide.jpg
exterior-front.jpg
bedroom-overview.jpg
kitchen-wide.jpg
dining-table.jpg
bathroom-main.jpg
access-road.jpg
logo.png
```

## Metadata To Track

For each image, track:

- filename
- public/private
- best use
- alt text
- source photographer
- date taken
- replacement needed: yes/no

## Agent Rules

The agent may:

- suggest where images should be used
- write alt text
- flag missing shots
- keep the website/gallery checklist current

The agent may not:

- publish private images without owner approval
- use misleading photos
- hide important property limitations
- invent photo descriptions
