# Contributing

Thanks for your interest in AgentBNB.

## Scope

- Keep contributions public-safe and clean-room.
- Do not include private Balay Pansol operations data, credentials, or customer details.
- Keep examples generic and reusable across properties.

## How to Contribute

1. Open an issue describing the problem or improvement.
2. Keep pull requests focused to one capability or documentation lane.
3. For feature PRs, include:
   - what changed
   - why it matters for operators
   - any migration or setup notes

## Priority Areas

- setup guides (Sheets, Resend, admin auth)
- clean-room templates for dashboard and agent workflow
- docs that improve first-time replication

## Code of Conduct

Be respectful, practical, and implementation-focused.
