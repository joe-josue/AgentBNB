# Website Template (White Label)

A clean-room website starter modeled after Balay’s working flow.

## Core Modules

- Public pages (hero, amenities, location, FAQs)
- Availability checker / booking calendar
- Pricing estimator
- Inquiry form
- Admin panel:
  - inquiries
  - bookings
  - notes/briefing
  - calendar view
- Feedback + Google Reviews flow

## Suggested Stack

- Next.js
- Tailwind CSS
- API routes for inquiry/admin actions
- Deploy on Vercel

## Outcome

A practical property site that is not brochure-only. It supports the full inquiry-to-booking-to-review loop.
