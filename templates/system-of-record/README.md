# System of Record Template

Use this as the property knowledge base and operational source of truth.

## Suggested Structure

```text
system-of-record/
  property-profile/
    property-overview.md
    amenities.md
    house-rules.md
    location-and-access.md
  operations/
    inquiry-scripts.md
    checkin-checkout-workflow.md
    escalation-playbook.md
    staff-handoff-template.md
  listings/
    airbnb-listing-copy.md
    booking-listing-copy.md
    direct-site-copy.md
  media/
    photos/
    videos/
  changes/
    changelog.md
```

## Why this matters

- Keeps website, listings, and guest responses consistent.
- Gives the agent complete context for accurate replies and updates.
- Makes operational updates easy to track and audit.
